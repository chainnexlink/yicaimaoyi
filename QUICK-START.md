## 第一步：安装必备插件

登录 WordPress 后台：http://182.92.10.131:21080/wp-admin

### 必装插件

| 插件 | 安装方法 |
|------|---------|
| Elementor | 后台搜索安装 |
| Astra | 外观→主题→添加 |
| Contact Form 7 | 后台搜索安装 |

### 推荐插件（可选）

- Elementor Pro（付费，功能更强）
- Yoast SEO（SEO优化）
- WP Super Cache（加速）
- Polylang（多语言）

---

## 第二步：激活主题

1. 外观 → 主题
2. 搜索 `Astra` 
3. 安装并激活

---

## 第三步：添加自定义CSS

1. 外观 → 自定义 → 额外CSS
2. 打开 `custom-styles.css` 文件
3. 复制全部内容粘贴进去
4. 点击「发布」保存

---

## 第四步：创建页面

### 4.1 创建所有页面

后台 → 页面 → 新建页面，依次创建：

| 页面名称 | 页面别名（URL）|
|---------|---------------|
| 首页 | home |
| 产品中心 | products |
| 关于我们 | about |
| 资质证书 | certifications |
| 新闻动态 | news |
| 联系我们 | contact |

### 4.2 设置首页

1. 设置 → 阅读
2. "您的主页显示" → 静态页面
3. 主页：选择「首页」
4. 文章页：选择「新闻动态」
5. 保存

### 4.3 创建导航菜单

1. 外观 → 菜单
2. 创建菜单名：主导航
3. 勾选所有页面 → 添加到菜单
4. 调整顺序：首页、产品中心、关于我们、资质证书、新闻动态、联系我们
5. 菜单位置：勾选「Primary Menu」
6. 保存菜单

---

## 第五步：使用 Elementor 编辑页面

### 编辑流程

1. 页面 → 所有页面
2. 找到要编辑的页面
3. 点击「使用 Elementor 编辑」
4. 参考 `pages/` 文件夹中对应的布局文档进行设计

### 常用操作

- 添加区块：点击 + 号
- 拖拽元素：从左侧面板拖入
- 修改样式：选中元素后点击「样式」选项卡
- 响应式预览：底部切换设备图标
- 保存：点击「更新」

## 常见问题

**Q: Elementor 加载很慢？**
A: 确保服务器配置足够，可安装缓存插件

**Q: 表单提交后收不到邮件？**
A: 安装 WP Mail SMTP 插件配置邮件发送

**Q: 手机端显示不正常？**
A: 在 Elementor 中切换到手机视图单独调整样式

**Q: 如何添加多语言？**
A: 安装 Polylang 插件，为每个页面创建不同语言版本
