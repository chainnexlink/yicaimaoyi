# 易采贸易网站搭建指南

## 一、基础环境配置

### 1.1 必装插件列表

登录 WordPress 后台 → 插件 → 安装插件，搜索并安装以下插件：

| 插件名称 | 用途 | 必选 |
|---------|------|------|
| Elementor | 可视化页面编辑器 | 是 |
| Elementor Pro | 高级功能（表单、主题构建） | 推荐 |
| Astra | 轻量级主题（与Elementor兼容性最佳） | 是 |
| WPForms Lite | 询盘表单（如无Elementor Pro） | 备选 |
| Contact Form 7 | 询盘表单备选方案 | 备选 |
| Polylang | 多语言支持 | 后续 |
| Yoast SEO | SEO优化 | 推荐 |
| WP Super Cache | 缓存加速 | 推荐 |

### 1.2 主题安装

1. 外观 → 主题 → 添加新主题
2. 搜索 `Astra` 并安装激活
3. 外观 → 自定义 → 进行基础配置

## 三、页面创建步骤

### 3.1 创建页面

1. 后台 → 页面 → 新建页面
2. 依次创建以下页面：
   - 首页
   - 产品中心
   - 关于我们
   - 资质证书
   - 新闻动态
   - 联系我们

### 3.2 设置首页

1. 设置 → 阅读
2. "您的主页显示" 选择 "静态页面"
3. 主页选择 "首页"
4. 文章页选择 "新闻动态"

### 3.3 创建导航菜单

1. 外观 → 菜单
2. 创建新菜单，命名为 "主导航"
3. 添加所有页面到菜单
4. 显示位置勾选 "Primary Menu"

---

## 四、Astra 主题配置

### 4.1 全局配置

外观 → 自定义，按以下设置：

**全局 → 排版**
- 基础字体：Noto Sans SC（中文）或 Source Sans Pro
- 基础字号：16px
- 标题字体：Noto Sans SC

**全局 → 颜色**
- 主色调：#1a5276（商务蓝）
- 链接颜色：#1a5276
- 链接悬停：#2980b9

**页眉 → 主页眉**
- 布局：Logo 左 + 菜单右
- 页眉宽度：全宽

**页脚**
- 页脚小部件：启用
- 版权文字：© 2024 易采贸易 版权所有

## 六、Elementor 页面编辑

### 6.1 使用 Elementor 编辑页面

1. 页面 → 所有页面
2. 选择要编辑的页面
3. 点击 "使用 Elementor 编辑"

### 6.2 页面布局说明

详见各页面的布局文档：
- `pages/home.md` - 首页布局
- `pages/products.md` - 产品中心布局
- `pages/about.md` - 关于我们布局
- `pages/certifications.md` - 资质证书布局
- `pages/news.md` - 新闻动态布局
- `pages/contact.md` - 联系我们布局
