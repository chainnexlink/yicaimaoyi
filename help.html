账户与认证

如何注册平台账户？

点击首页右上角"登录/注册"按钮，填写手机号、设置密码，完成验证码验证即可注册。企业用户需完成企业认证后才能使用全部功能。

采购商和供应商有什么区别？

采购商（用户）可发布采购需求、接收报价、管理订单；供应商可发布产品、响应需求、管理生产。一个账户可以同时拥有两种角色，通过"个人中心"和"供应商中心"分别管理。

如何完成企业资质认证？

进入"资质认证"页面，点击"申请认证"，按步骤填写企业基本信息、法人信息、上传营业执照等资质证书。提交后平台将在1-3个工作日内完成审核。认证通过后可享受优先匹配、信任徽章等权益。

供应商如何上传资质证书？

供应商可在"供应商中心 → 个人信息 → 资质认证管理"中上传ISO、CE等行业认证证书。上传后平台会进行审核验证，通过后资质信息将展示在供应商主页，提高匹配权重和信任度。

认证过期了怎么办？

认证到期前30天系统会发送提醒通知。您可以在"资质认证"页面找到对应认证，点击"续期"按钮重新上传最新的证书文件进行续期。供应商也可在供应商中心的认证管理中完成续期操作。

订单与交易

如何发布采购需求？

在"个人中心 → 需求管理 → 发布需求"中填写需求标题、品类、数量、预算等信息后发布。平台会自动匹配合适的供应商并推送报价。

订单产生纠纷怎么办？

1. 首先通过消息中心与供应商直接沟通协商
2. 若协商不成，可在客服中心联系"纠纷调解专员"
3. 平台将在48小时内介入调解
4. 严重问题可提交工单，平台会安排专人跟进处理

如何查看订单的生产进度？

进入"个人中心 → 我的订单"，点击对应订单的"生产进度"按钮即可查看。供应商会通过供应商中心的"监控上传"功能定期更新生产进度，确保您实时了解生产状态。

合同如何签署和管理？

接受报价后系统自动生成电子合同。在"个人中心 → 合同管理 → 待签合同"中查看并签署。签署后可查看履约进度、申请变更或终止。

供应商合作

如何申请成为供应商？

在登录页面选择"供应商入驻"，填写企业信息、上传营业执照和相关资质证书，选择主营品类后提交申请。平台会在1-3个工作日内完成审核。审核通过后即可使用供应商中心管理产品和订单。

供应商如何提高匹配权重？

1. 完善企业信息和产品资料
2. 上传完整的资质认证证书（ISO、CE等）
3. 保持较高的履约完成率
4. 积极响应采购需求和询盘
5. 获取好评提高信用评分

供应商需要上传哪些资质？

必需：营业执照。推荐上传：ISO9001质量管理认证、ISO14001环境管理认证、行业特种许可证、CE/UL等产品认证。更多资质认证可提高供应商信任度和匹配优先级。

平台功能

智能匹配是如何工作的？

平台根据采购需求的品类、数量、预算、地域等维度，结合供应商的产品、资质、信用评分、履约率等数据，通过AI算法进行多维度匹配，推荐最适合的供应商。

消息通知有哪些类型？

包括：订单状态通知、生产进度更新、物流信息、认证审核结果、报价提醒、合同待签通知、预警信息等。可在"消息中心"中统一查看和管理。

三方沟通是如何进行的？

平台支持采购商、供应商、平台客服三方实时沟通：
1. 采购商可在"个人中心"或"客服中心"直接联系供应商
2. 供应商可在"供应商中心"回复采购商消息
3. 任一方可在客服中心邀请平台介入协调
4. 平台客服可在纠纷中同时与双方沟通协调

账户与认证 更新于 2026-02-10

如何完成企业资质认证？

企业资质认证是使用平台全部功能的前提。完成认证后可享受优先匹配、信任徽章等权益。

一、认证入口

  • 方式1：首页导航栏 → 资质认证
  • 方式2：个人中心 → 企业信息 → 去认证
  • 方式3：供应商中心 → 资质认证管理

二、认证步骤

第1步：填写企业基本信息

  • 企业名称（需与营业执照一致）
  • 统一社会信用代码
  • 法定代表人姓名及身份证号
  • 企业注册地址、联系电话

第2步：上传资质证书

  • 必需：营业执照扫描件（清晰彩色照片）
  • 推荐：ISO9001质量管理认证
  • 推荐：行业特种经营许可证
  • 推荐：产品认证（CE/UL/FDA等）
提示：上传更多资质证书可以提高供应商匹配权重和平台信任度评分。

第3步：提交审核

确认信息无误后点击"提交认证"，平台将在1-3个工作日内完成审核。审核过程中可在"资质认证"页面查看进度。

三、审核结果

  • 通过：获得认证徽章，享受全平台功能
  • 补充材料：按提示补充缺失材料后重新提交
  • 驳回：查看驳回原因，修正后可重新申请
常见问题：如果营业执照发生变更，请先在工商部门完成变更登记，再上传新的营业执照进行认证更新。

四、认证续期

认证到期前30天系统会自动发送提醒。在"资质认证"页面找到对应认证点击"续期"即可。供应商可在供应商中心的认证管理中统一管理所有认证续期。
