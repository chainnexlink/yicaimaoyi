## 详细搭建步骤

### Section 1: Hero 横幅区域

**Elementor 操作步骤：**
1. 添加新区块（Section）
2. 布局 → 内容宽度：全宽
3. 样式 → 背景：
   - 类型：经典
   - 颜色：渐变（#1a5276 → #154360）
   - 或上传企业形象大图
4. 样式 → 背景叠加：颜色 rgba(26,82,118,0.85)
5. 高级 → 内边距：上100px 下100px

**添加元素：**
- 标题（Heading）：
  ```
  您的一站式国际贸易采购伙伴
  ```
  - 标签：H1
  - 对齐：居中
  - 样式：颜色 #FFFFFF，字号 48px，粗细 700

- 文本编辑器：
  ```
  专注于优质产品采购与出口服务，连接中国制造与全球市场
  ```
  - 颜色：rgba(255,255,255,0.9)
  - 字号：18px
  - 对齐：居中

- 按钮组（Button）：
  - 按钮1：`浏览产品` → 链接到产品中心
  - 按钮2：`联系我们` → 链接到联系页面
  - 样式：主按钮 + 边框按钮

---

### Section 2: 快速入口/搜索

**Elementor 操作步骤：**
1. 添加新区块
2. 背景：#f8f9fa（浅灰）
3. 内边距：上30px 下30px

**添加元素：**
- 图标列表或按钮组，展示热门分类：
  - 电子产品
  - 机械设备
  - 家居用品
  - 服装纺织
  - 查看全部 →

---

### Section 3: 核心优势

**Elementor 操作步骤：**
1. 添加新区块
2. 结构：4列等宽
3. 内边距：上60px 下60px

**每列添加图标盒子（Icon Box）：**

| 图标 | 标题 | 描述 |
|------|------|------|
| fa-certificate | 品质保证 | 严格质检，确保每件产品符合国际标准 |
| fa-dollar-sign | 价格优势 | 工厂直采，为您节省中间成本 |
| fa-clock | 快速响应 | 24小时内报价，专业团队全程跟进 |
| fa-globe | 全球配送 | 覆盖100+国家，灵活物流方案 |

**图标盒子样式设置：**
- 图标位置：顶部
- 图标大小：60px
- 图标颜色：#1a5276
- 图标背景：圆形，#f0f4f7
- 标题颜色：#2c3e50
- 描述颜色：#5d6d7e

---

### Section 4: 热门产品

**Elementor 操作步骤：**
1. 添加新区块
2. 内边距：上60px 下60px

**添加元素：**
- 标题：`热门产品`
  - 添加 CSS 类：`section-title text-center`
  
- 使用「文章」小部件或「图库」：
  - 布局：3列网格
  - 间距：30px
  - 显示6-8个产品

**产品卡片结构（使用内部区块）：**
```
┌──────────────────┐
│   [产品图片]      │
├──────────────────┤
│   产品名称        │
│   产品分类        │
│   [了解详情]      │
└──────────────────┘
```

---

### Section 5: 公司简介

**Elementor 操作步骤：**
1. 添加新区块
2. 结构：2列（左60% 右40%）
3. 背景：#f8f9fa
4. 内边距：上80px 下80px

**左侧内容：**
- 小标题：`关于易采贸易`
- 主标题：`您值得信赖的国际贸易伙伴`
- 段落文字：公司介绍（150-200字）
- 按钮：`了解更多` → 链接到关于我们

**右侧内容：**
- 图片：公司大楼/团队/仓库照片
- 或视频：企业宣传片

---

### Section 6: 数据统计

**Elementor 操作步骤：**
1. 添加新区块
2. 背景：#1a5276（商务蓝）
3. 结构：4列
4. 内边距：上50px 下50px

**每列添加计数器（Counter）：**

| 数字 | 后缀 | 标题 |
|------|------|------|
| 15 | + | 年行业经验 |
| 2000 | + | 服务客户 |
| 100 | + | 出口国家 |
| 5000 | + | 产品种类 |

**计数器样式：**
- 数字颜色：#FFFFFF
- 数字大小：48px
- 标题颜色：rgba(255,255,255,0.8)

---

### Section 7: 合作客户

**Elementor 操作步骤：**
1. 添加新区块
2. 内边距：上40px 下40px

**添加元素：**
- 标题：`合作伙伴`
- 图像轮播（Image Carousel）：
  - 上传客户Logo图片
  - 每行显示：5-6个
  - 自动播放：开启
  - 导航：隐藏
  - 无限循环：开启

---

### Section 8: 最新动态

**Elementor 操作步骤：**
1. 添加新区块
2. 内边距：上60px 下60px

**添加元素：**
- 标题：`新闻动态`
- 文章小部件（Posts）：
  - 布局：经典
  - 列数：3
  - 显示数量：3
  - 显示：标题、日期、摘要
  - 隐藏：作者、分类

---

### Section 9: CTA 行动召唤

**Elementor 操作步骤：**
1. 添加新区块
2. 背景：渐变（#1a5276 → #2980b9）
3. 内边距：上50px 下50px
4. 文字对齐：居中

**添加元素：**
- 标题：`准备好开始您的采购之旅了吗？`
  - 颜色：#FFFFFF
- 按钮：`立即询盘`
  - 尺寸：大
  - 背景：#FFFFFF
  - 文字：#1a5276
  - 链接：联系我们页面
