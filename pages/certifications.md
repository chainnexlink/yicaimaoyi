## 详细搭建步骤

### Section 1: 页面横幅

**Elementor 设置：**
- 最小高度：250px
- 背景：#1a5276 或相关背景图
- 叠加：rgba(26, 82, 118, 0.85)

**内容：**
- 标题：`资质证书`（H1，白色，42px）
- 面包屑：`首页 > 资质证书`

---

### Section 2: 简介说明

**布局：单列**
**内边距：上40px 下40px**
**最大宽度：800px，居中**

**内容：**
```
易采贸易始终坚持合规经营，严格遵守国内外贸易法规。公司已获得多项权威认证和资质，为客户提供品质保障和信任基础。以下是我们的主要资质证书和荣誉认证。
```

---

### Section 3: 企业资质

**布局：单列**
**背景：#f8f9fa**
**内边距：上60px 下60px**

**标题：** `企业资质` + 装饰线

**证书网格（3列）：**
```
┌────────────────┐  ┌────────────────┐  ┌────────────────┐
│                │  │                │  │                │
│   [证书图片]    │  │   [证书图片]    │  │   [证书图片]    │
│                │  │                │  │                │
├────────────────┤  ├────────────────┤  ├────────────────┤
│   营业执照      │  │  进出口经营权   │  │   海关注册      │
│   2008年取得    │  │   2009年取得   │  │   2009年取得    │
└────────────────┘  └────────────────┘  └────────────────┘
```

**使用内部区块（Inner Section）构建卡片：**
- 图片小部件：上传证书图片
- 标题：证书名称
- 文本：获得时间/有效期

**卡片样式：**
- 背景：白色
- 内边距：20px
- 圆角：8px
- 阴影：0 2px 10px rgba(0,0,0,0.05)

---

### Section 4: 体系认证

**布局：单列**
**背景：白色**
**内边距：上60px 下60px**

**标题：** `体系认证`

**证书网格（4列）：**

| 认证名称 | 说明 |
|---------|------|
| ISO 9001 | 质量管理体系认证 |
| ISO 14001 | 环境管理体系认证 |
| SGS | 产品检测认证 |
| CE | 欧盟产品认证 |

**展示方式 - 认证卡片：**
```
┌──────────────────┐
│   [认证Logo]      │
│                  │
│   ISO 9001       │
│   质量管理体系    │
│   认证编号：XXX   │
└──────────────────┘
```

**卡片样式：**
- 边框：1px solid #e0e0e0
- 悬停：边框变蓝 + 轻微上浮

---

### Section 5: 荣誉奖项

**布局：单列**
**背景：#f8f9fa**
**内边距：上60px 下60px**

**标题：** `荣誉奖项`

**荣誉展示（时间轴或网格形式）：**

```
2023年 ──── 年度优秀外贸企业
2022年 ──── 诚信经营示范单位
2021年 ──── 出口贡献奖
2020年 ──── 行业十佳供应商
```

或使用图片画廊展示奖杯/奖状照片

---

### Section 6: 合作伙伴

**布局：单列**
**内边距：上40px 下40px**

**标题：** `合作伙伴`

**使用图像轮播展示合作品牌Logo：**
- 每行显示：5-6个
- 自动播放
- Logo灰度处理，悬停恢复彩色

**Logo样式CSS：**
```css
.partner-logo img {
    filter: grayscale(100%);
    opacity: 0.6;
    transition: all 0.3s ease;
}

.partner-logo img:hover {
    filter: grayscale(0%);
    opacity: 1;
}
```

## 需要准备的素材

- [ ] 营业执照扫描件（可打码处理敏感信息）

- [ ] SGS/CE等产品认证
- [ ] 荣誉证书/奖状照片

**图片建议：**
- 格式：JPG或PNG
- 尺寸：800x600px 左右
- 清晰度：确保文字可读
- 隐私处理：敏感信息打码
