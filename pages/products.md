## 详细搭建步骤

### 准备工作：创建产品分类

**后台 → 文章 → 分类目录**（或使用自定义文章类型）

建议分类结构：
```
产品中心
├── 电子产品
│   ├── 消费电子
│   └── 电子元器件
├── 机械设备
│   ├── 工业机械
│   └── 农业机械
├── 家居用品
│   ├── 家具
│   └── 家装材料
├── 服装纺织
│   ├── 服装
│   └── 面料辅料
└── 其他产品
```

---

### Section 1: 页面横幅

**Elementor 操作步骤：**
1. 添加新区块
2. 高度：最小高度 250px
3. 背景：渐变 #1a5276 → #154360（或产品相关图片+叠加）
4. 内容垂直对齐：居中

**添加元素：**
- 标题：`产品中心`
  - H1，颜色：#FFFFFF
  - 字号：42px
  
- 面包屑导航（文本编辑器）：
  ```
  首页 > 产品中心
  ```
  - 颜色：rgba(255,255,255,0.7)
  - 字号：14px

---

### Section 2: 产品分类导航

**Elementor 操作步骤：**
1. 添加新区块
2. 背景：#f8f9fa
3. 内边距：上20px 下20px

**添加元素 - 按钮组或标签：**
- 全部产品
- 电子产品
- 机械设备
- 家居用品
- 服装纺织
- 更多分类 ▼

**按钮样式：**
- 背景：透明
- 边框：1px solid #ddd
- 文字：#5d6d7e
- 悬停：背景 #1a5276，文字 #fff
- 圆角：20px
- 内边距：8px 20px

---

### Section 3: 产品网格

**Elementor 操作步骤：**
1. 添加新区块
2. 内边距：上40px 下40px

**方案A：使用文章小部件（Posts Widget）**
- 皮肤：经典
- 列数：4（桌面），2（平板），1（手机）
- 间距：30px
- 图片位置：顶部
- 图片比例：4:3
- 显示：标题、分类
- 隐藏：作者、日期、评论

**方案B：使用图库+内部区块（手动排版）**

每个产品卡片结构：
```html
<div class="product-card">
  <div class="product-image">
    <img src="产品图片" alt="产品名称">
    <div class="product-overlay">
      <a href="产品详情" class="btn-view">查看详情</a>
      <a href="询盘页面" class="btn-inquiry">立即询盘</a>
    </div>
  </div>
  <div class="product-content">
    <span class="product-category">产品分类</span>
    <h3 class="product-title">产品名称</h3>
    <p class="product-desc">简短描述...</p>
  </div>
</div>
```

---

### 产品卡片悬停效果 CSS

在自定义CSS中已包含，确保添加以下类名：

```css
/* 产品卡片悬停显示操作按钮 */
.product-card .product-overlay {
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background: rgba(26, 82, 118, 0.9);
    display: flex;
    flex-direction: column;
    align-items: center;
    justify-content: center;
    opacity: 0;
    transition: opacity 0.3s ease;
}

.product-card:hover .product-overlay {
    opacity: 1;
}

.product-overlay .btn-view,
.product-overlay .btn-inquiry {
    display: block;
    padding: 10px 25px;
    margin: 5px 0;
    color: #fff;
    text-decoration: none;
    border: 2px solid #fff;
    border-radius: 4px;
    transition: all 0.3s ease;
}

.product-overlay .btn-view:hover {
    background: #fff;
    color: #1a5276;
}

.product-overlay .btn-inquiry {
    background: #e67e22;
    border-color: #e67e22;
}
```

### Section 4: 分页导航

**Elementor 操作步骤：**
1. 如使用文章小部件，分页会自动显示
2. 如手动排版，添加分页HTML

**分页样式CSS（已包含在主样式中）：**
```css
.pagination {
    display: flex;
    justify-content: center;
    gap: 10px;
    margin-top: 40px;
}

.pagination a,
.pagination span {
    padding: 10px 15px;
    border: 1px solid #ddd;
    color: #5d6d7e;
    text-decoration: none;
    border-radius: 4px;
    transition: all 0.3s ease;
}

.pagination a:hover,
.pagination .current {
    background: #1a5276;
    color: #fff;
    border-color: #1a5276;
}
```

## 产品详情页模板

如需创建单独的产品详情页，结构如下：

```
┌─────────────────────────────────────────────────────┐
│  面包屑：首页 > 产品中心 > 分类 > 产品名称           │
└─────────────────────────────────────────────────────┘
┌────────────────────┬────────────────────────────────┐
│                    │   产品名称                      │
│   产品图片画廊      │   产品编号：XXX-001            │
│   （主图+缩略图）   │   产品分类：电子产品            │
│                    │   产品描述：...                 │
│                    │   [立即询盘] [添加到询盘单]      │
└────────────────────┴────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│  产品详情 | 规格参数 | 包装运输                       │
├─────────────────────────────────────────────────────┤
│  详细的产品描述内容...                               │
│  产品参数表格...                                     │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
```
