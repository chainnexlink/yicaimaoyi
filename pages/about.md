# 关于我们 Elementor 布局指南

## 页面结构概览

```
┌─────────────────────────────────────────────────────┐
│                   页面横幅                           │
│  "关于我们" 标题 + 企业口号                          │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│                   公司简介                           │
│  左侧：公司介绍文字 | 右侧：公司图片/视频             │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│                   发展历程                           │
│  时间轴形式展示公司里程碑                            │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│                   企业文化/价值观                     │
│  使命 | 愿景 | 价值观 三栏展示                        │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│                   数据展示                           │
│  成立年份 | 员工人数 | 服务客户 | 出口国家            │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│                   团队介绍                           │
│  核心团队成员展示                                    │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│                   办公环境/仓库展示                   │
│  图片画廊                                            │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│                   CTA                                │
│  "与我们合作" 按钮                                   │
└─────────────────────────────────────────────────────┘
```

---

## 详细搭建步骤

### Section 1: 页面横幅

**Elementor 设置：**
- 最小高度：300px
- 背景：企业形象图片 + 深色叠加
- 叠加颜色：rgba(26, 82, 118, 0.8)

**内容：**
- 标题：`关于易采贸易`（H1，白色，48px）
- 副标题：`您值得信赖的国际贸易合作伙伴`（白色，18px）

---

### Section 2: 公司简介

**布局：2列（55% | 45%）**
**背景：白色**
**内边距：上80px 下80px**

**左列内容：**
```
小标题：公司简介
主标题：专注国际贸易 连接全球市场

[替换为您的公司介绍，建议200-300字]

易采贸易有限公司成立于2008年，是一家专注于国际贸易采购与出口服务的综合性贸易公司。公司总部位于[城市]，拥有专业的采购团队、严格的质量控制体系和完善的物流配送网络。

多年来，我们始终秉承"诚信为本、品质至上"的经营理念，致力于为全球客户提供优质的中国制造产品。我们的业务涵盖电子产品、机械设备、家居用品、服装纺织等多个领域，产品远销欧美、东南亚、中东等100多个国家和地区。

凭借丰富的行业经验、广泛的供应商网络和专业的服务团队，易采贸易已成为众多海外采购商值得信赖的合作伙伴。
```

- 按钮：`了解我们的服务` → 链接到产品中心

**右列内容：**
- 图片：公司大楼或团队合照
- 或嵌入企业宣传视频

---

### Section 3: 发展历程（时间轴）

**布局：单列**
**背景：#f8f9fa**
**内边距：上60px 下60px**

**标题：** `发展历程`

**时间轴内容示例：**

```
2008年 ────●──── 公司成立
              在[城市]注册成立，开始国际贸易业务

2012年 ────●──── 业务拓展
              开拓东南亚市场，年出口额突破500万美元

2015年 ────●──── 团队壮大
              员工人数突破50人，建立专业质检团队

2018年 ────●──── 十周年庆
              服务客户突破1000家，成为行业标杆企业

2020年 ────●──── 数字化转型
              上线B2B电商平台，开启线上线下融合发展

2024年 ────●──── 持续发展
              年出口额突破1亿美元，业务覆盖100+国家
```

**Elementor 实现方式：**
使用「时间线」小部件（Elementor Pro）或手动排版

---

### Section 4: 企业文化

**布局：3列**
**背景：白色**
**内边距：上60px 下60px**

**每列使用图标盒子（Icon Box）：**

| 图标 | 标题 | 内容 |
|------|------|------|
| fa-bullseye | 企业使命 | 连接中国制造与全球市场，为客户创造价值 |
| fa-eye | 企业愿景 | 成为全球最受信赖的国际贸易服务商 |
| fa-heart | 核心价值观 | 诚信、专业、创新、共赢 |

**图标盒子样式：**
- 背景：#f0f4f7
- 内边距：30px
- 圆角：8px
- 图标颜色：#1a5276
- 居中对齐

---

### Section 5: 数据展示

**布局：4列**
**背景：#1a5276**
**内边距：上50px 下50px**

**使用计数器（Counter）：**

| 数字 | 后缀 | 标题 |
|------|------|------|
| 2008 | 年 | 公司成立 |
| 100 | + | 专业员工 |
| 2000 | + | 服务客户 |
| 100 | + | 出口国家 |

**计数器样式：**
- 数字颜色：#FFFFFF
- 数字大小：56px
- 标题颜色：rgba(255,255,255,0.8)

---

### Section 6: 团队介绍（可选）

**布局：4列**
**内边距：上60px 下60px**

**标题：** `核心团队`

**每列内容：**
```
┌──────────────┐
│   [头像]      │
│   姓名        │
│   职位        │
│   简介...     │
└──────────────┘
```

**示例：**
- 张总 - 创始人兼CEO
- 李经理 - 销售总监
- 王经理 - 采购总监
- 陈经理 - 质量总监

---

### Section 7: 办公环境展示

**布局：单列**
**背景：#f8f9fa**
**内边距：上60px 下60px**

**标题：** `办公环境`

**使用图库（Gallery）或图像轮播：**
- 办公室照片
- 仓库照片
- 质检中心照片
- 团队活动照片

**画廊设置：**
- 列数：4
- 间距：10px
- 点击：灯箱放大

---

### Section 8: CTA

**布局：单列**
**背景：渐变 #1a5276 → #2980b9**
**内边距：上50px 下50px**
**居中对齐**

**内容：**
- 标题：`准备与我们合作？`（白色）
- 副标题：`无论您是寻找供应商还是希望拓展国际市场，我们都能为您提供专业服务`
- 按钮：`联系我们` → 链接到联系页面

- [ ] 公司简介文字（200-300字）

- [ ] 公司数据（员工数、客户数等）
- [ ] 企业使命、愿景、价值观
- [ ] 公司图片（大楼、办公室、仓库）
- [ ] 团队成员照片和介绍（可选）
- [ ] 企业宣传视频（可选）
